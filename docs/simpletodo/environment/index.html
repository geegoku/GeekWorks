    Djangoでプロジェクトの作成

    プロジェクトを作成したい場所に移動して、以下コマンドを実行してください。

    1
2
3
    django-admin startproject simpletodo
cd simpletodo
python manage.py startapp todo

    django-admin startproject プロジェクト名 は、プロジェクトを作成するコマンドです。
    python manage.py startapp アプリ名 は、アプリを作成するコマンドです。

    プロジェクトとアプリの違いについて

    Djangoでは1つのプロジェクトに対して、複数のアプリを入れることができます。
    ここで言うプロジェクトは「サイト」、アプリが「機能」と考えてください。
    例えば、facebookがDjangoで作られていた場合、以下のような構成になります。

    facebook (プロジェクト)
       users (アプリ)
       events (アプリ)
       messages (アプリ)
       ...

    Warning

    プロジェクト名とアプリ名には、小文字の半角英数字のみを使用してください。
    ハイフン「-」やアンダーバー「_」は、使わないでください。(使ってしまうと、後々めんどいです)

    django-admin startproject simple-todo (ダメな例)
    python manage.py startapp to_do　(ダメな例)

    コマンドで、Djangoを起動します。 

    1
    python manage.py runserver

    localhostにアクセスして、Djangoが起動していたら完了です。

    http://localhost:8000/
